# Equity-Valuation-Report-Infineon-

Equity valuation project combining a fully linked Excel DCF model with an integrated investment memo and clear Buy/Hold/Sell recommendation for Infineon Technologies AG (IFX).

Project overview
This project was built to demonstrate practical equity research skills: financial statement analysis, forecasting, discounted cash flow (DCF) valuation, and written communication of an investment thesis, all in a single Excel file.

It is designed as a portfolio piece for applications in equity research, asset management, and investment banking roles.

Key features
Fully linked Excel DCF model with explicit forecast period and terminal value calculation.
Scenario analysis (Base / Bear / Bull) controlled by a scenario selector cell and a dedicated scenarios sheet.
Clear separation of inputs, calculations, and outputs across dedicated tabs (Assumptions, Historical_Financials, Forecast, DCF, Scenarios, Sensitivities, Summary/Memo).
Equity value and implied price per share, including a cost of equity assumption around 9.5% and implied 2035 value per share of approximately 34.28 EUR.
Integrated investment memo sheet summarizing business overview, key drivers, valuation results, and a Buy/Hold/Hold recommendation.

Repository structure
DCF_Infineon.xlsx – Main Excel file containing:
  Input and assumptions sheets.
  Historical and forecast financial statements.
  DCF valuation, scenarios, sensitivities, and key outputs.

Methodology
Forecasting: Free cash flows are projected from historical performance, using explicit assumptions for revenue growth, operating margins, tax rate, capex, and working capital needs.

Discount rate: The model applies a cost of equity / WACC consistent with CAPM-based inputs and Infineon’s target capital structure.

Terminal value: Terminal value is estimated using a perpetual growth approach with a conservative long‑term growth rate not exceeding long‑run GDP growth.

Sensitivity & scenarios: The model includes scenario and sensitivity analysis on key drivers such as WACC, terminal growth, and margin assumptions.

How to use
Download and open DCF_Infineon.xlsx.
Start in the Assumptions sheet to review or adjust key inputs (WACC, terminal growth, tax rate, growth assumptions).
Use the scenario selector cell on the Assumptions sheet to switch between Base, Bear, and Bull cases and see the impact on value per share in the DCF and Summary outputs.
Go to the Summary/Memo sheet to read the integrated investment memo and see the high-level conclusion of the valuation.
